# Database

In this guide I will be explaining how to install MariaDB in combination with HeidiSQL to manage your server's database.

## MariaDB
<Steps>
### Download MariaDB

- Download MariaDB from [the official website](https://mariadb.org/download)
- Select MariaDB Server Version **10.11.8**
- Select "Windows" as "Operating System"
- The website should now look like <LinkButton text={"this"} image={"https://r2.fivemanage.com/pub/kx9r76c09t4h.png"}/>
- Leave the "mirror" default
- Press "Download"

### Install MariaDB

Now that we have installed the MSI file, its time to install the application.
- Run the MSI file
- Press next
- Tick the box for License Agreement and press next
- Leave the settings default, press next
- You can either choose to untick the box to continue without a password or create a password, that is up to you. If you have done that press Next again
- Leave all the settings default and press next
- Press "Install"
- If your computer asks you "Do you want this app to make changes to your device" click "Yes".
- Once that is done you should be able to click "Finish".
- To make sure that MariaDB is properly installed, we will go to <LinkButton text={"services.msc"} image={"https://r2.fivemanage.com/pub/a5802720e572.png"}/> and look for "MariaDB"
- If you followed all these steps, MariaDB should be present in the services like <LinkButton text={"this"} image={"https://r2.fivemanage.com/pub/x1x2bxnlkey4.png"}/>
</Steps>

## HeidiSQL

<Steps>

### Download HeidiSQL

- Download HeidiSQL from [the official website](https://www.heidisql.com/download.php).
- Click <LinkButton text={"Installer, 32/64 bit combined"} image={"https://r2.fivemanage.com/pub/sfb32sojf0hx.png"}/>.
- Save the executable (.exe) to your download location and press "Save"

### Install HeidiSQL

- Open the just installed .exe file.
- If <LinkButton text={"this window"} image={"https://r2.fivemanage.com/pub/9lbtinbwwr8x.png"}/> appears, press "Install for all users".
- If your computer asks you "Do you want this app to make changes to your device" click "Yes".
- Select the option for <LinkButton text={"I accept the agreement"} image={"https://r2.fivemanage.com/pub/56dxde644zq5.png"}/> and press next.
- Leave the <LinkButton text={"Destination Location"} image={"https://r2.fivemanage.com/pub/o9rgnynnca4j.png"}/> default and press next.
- Press next again on the <LinkButton text={"Select Start Menu Folder"} image={"https://r2.fivemanage.com/pub/tcc6hdfz7lph.png"}/> option.
- For the <LinkButton text={"Select Additional Tasks"} image={"https://r2.fivemanage.com/pub/43g9s56nqriu.png"}/> you can decide what options you want to turn on. Press next again.
- Press "Install"
- Click "Finish"

</Steps>
